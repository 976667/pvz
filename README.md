# PVZ - 植物大战僵尸

基于 C++/Qt 实现的植物大战僵尸简化版游戏。

## 项目简介

这是一个使用 Qt Graphics View 框架开发的植物大战僵尸游戏，包含完整的植物系统、僵尸系统、卡牌系统、天气系统和游戏平衡机制。

## 游戏功能

### 植物系统
| 植物 | 功能 |
|------|------|
| 向日葵 | 生产阳光 |
| 豌豆射手 | 发射豌豆攻击僵尸 |
| 寒冰射手 | 发射冰豌豆，减速僵尸 |
| 双枪射手 | 连续发射两颗豌豆 |
| 坚果墙 | 高生命值防御植物 |
| 樱桃炸弹 | 范围爆炸伤害 |
| 土豆雷 | 延时爆炸攻击 |

### 僵尸系统
| 僵尸 | 特点 |
|------|------|
| 普通僵尸 | 基础属性 |
| 路障僵尸 | 较高生命值 |
| 铁桶僵尸 | 高生命值 |
| 读报僵尸 | 中等生命值 |
| 橄榄球僵尸 | 高生命值+快速移动 |

### 天气系统
- 晴天、多云、雨天、雪天、大风、沙尘暴
- 天气每2分钟切换一次（随游戏倍速变化）
- 不同天气影响植物和僵尸的行为

### 卡片系统
- **阳光卡**：立即获得200阳光，冷却2分钟
- **加速卡**：2倍速持续5秒，冷却2分钟
- **卡片升级**：双击卡片升级，需要1000阳光

### 升级机制
- **豌豆射手**：等级提升增加伤害（倍率0.5/级）
- **向日葵**：等级提升减少生产间隔
- **坚果墙**：等级提升增加生命值（倍率0.5/级），升级后回复满血
- 植物每次升级后回复至最大生命值
- 每次最多升一级（不会连续升级）
- 僵尸根据植物平均等级获得平衡性增强

### 僵尸刷新规则
- 0-1分钟：普通僵尸、路障僵尸
- 1-2分钟：增加铁桶僵尸
- 2-3分钟：增加读报僵尸
- 3分钟后：增加橄榄球僵尸

## 编译运行

### 环境要求
- Qt 5.x 或更高版本
- C++11 编译器

### 编译步骤
```bash
# 使用 qmake 构建
qmake PVZ.pro
make

# 或使用 Qt Creator 打开 .pro 文件直接运行
```

## 操作说明

1. **放置植物**：点击卡牌选择，然后点击草坪格子放置
2. **使用道具**：点击底部阳光卡或加速卡使用
3. **铲除植物**：点击铲子图标，再点击要铲除的植物
4. **调整倍速**：使用右上角按钮调整游戏速度
5. **升级卡片**：双击底部卡片进行升级（需要1000阳光）

## 游戏平衡

### 初始设置
- **初始阳光**：150
- **阳光卡**：200阳光
- **天空掉落阳光**：25/个
- **卡片升级费用**：1000阳光

### 植物属性
| 植物 | 生命值 | 攻击力/冷却 |
|------|-------|-----------|
| 向日葵 | 300 | 10秒/次 |
| 豌豆射手 | 300 | 25伤害/1.4秒 |
| 坚果墙 | 4000（Lv1）/ 6000（Lv2）/ 8000（Lv3） | - |

### 坚果墙升级机制
| 项目 | 数值 |
|------|------|
| 被动经验 | 每60帧获得4点经验 |
| 受伤经验 | damageTaken / 50（取整） |
| 1→2级所需经验 | 500点 |
| 2→3级所需经验 | 1500点 |
| 升级效果 | 生命值×1.5，回复满血 |

### 坚果墙外观状态
| 血量范围 | 外观 |
|---------|------|
| > 60% | WallNut.gif（满血） |
| 30% ~ 60% | WallNut1.gif（低血量） |
| ≤ 30% | WallNut2.gif（濒死） |

### 僵尸属性
| 僵尸 | 生命值 | 攻击力 |
|------|-------|-------|
| 普通 | 200 | 1 |
| 路障 | 480 | 1 |
| 铁桶 | 1030 | 1 |
| 读报 | 1030 | 1 |
| 橄榄球 | 1250 | 1 |

## 技术特点

- 使用 Qt Graphics View 实现游戏渲染
- 基于帧的游戏循环（~30 FPS）
- 碰撞检测和状态机管理
- 面向对象设计，植物和僵尸继承自通用基类
